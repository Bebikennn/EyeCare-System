# Use Case Diagrams (Separated)

## Overview
The EyeCare system has two main applications in this workspace:
- **App3**: Flutter mobile app + Flask backend API
- **eyecare_admin**: Admin dashboard (analytics, user management, model operations)

To keep documentation clear, the use case diagram is separated by application.

## Diagrams (PlantUML)
- App3 (mobile + backend): [use_case_diagram.puml](use_case_diagram.puml)
- Admin dashboard: [eyecare_admin_use_case_diagram.puml](eyecare_admin_use_case_diagram.puml)

To render:
- VS Code: install a PlantUML extension and preview the `.puml` files.
- CLI (optional): use PlantUML/JAR, or an online PlantUML server.
